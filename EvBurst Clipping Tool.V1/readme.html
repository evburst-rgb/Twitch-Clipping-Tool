Welcome

Thank you for downloading the EvBurst Clipping Tool. This application allows you to instantly create Twitch clips, trigger clips directly from Stream Deck, automatically post clips into Twitch chat, and view your recent clips directly from the dashboard.

Quick Setup

  1. Launch EvBurst Clipping Tool.exe
  2. Login with your Twitch account
  3. Authorize the requested Twitch permissions
  4. Copy your Stream Deck Trigger URL
  5. Add a Website action inside Stream Deck
  6. Paste the Trigger URL into the Stream Deck action
  7. Press the button while live to instantly create clips

Features

  • One-button Stream Deck clipping
  • Automatic Twitch chat posting
  • Persistent clip history dashboard
  • Cloud hosted backend system
  • Auto-refresh Twitch token support
  • Fast deployment-ready architecture

Important

Your Twitch channel MUST be live for clipping to work. Do not share your Stream Deck Trigger URL with anyone. Your trigger URL functions like a private API key.
